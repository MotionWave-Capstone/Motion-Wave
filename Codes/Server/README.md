## Motion Wave
### API Server 구축하기
- 자동차 성능 비교를 위한 데이터 값 전달

### 실행 방법
- 설정

    ```
    -- DB 연결 설정
    npm install mariadb
    ```
    
    ```
    -- swagger 연결 설정
    npm install swagger-jsdoc swagger-ui-express --save-dev
    ```

- 실행
    ```
    node app.js
    ```
    ```
    -- 확인 가능
    http://localhost:3000/api-docs
    ```
